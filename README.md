# 📊 Profit Analysis Dashboard

Interactive business intelligence dashboard developed using Looker Studio to analyze sales performance, profitability trends, regional contribution, shipping performance, and category-wise business insights across the United States.

## 🔗 Live Dashboard
👉 [View Interactive Dashboard](https://datastudio.google.com/reporting/e986ee0d-0e79-47b5-a654-a3f532456e38)

## 📌 Dashboard Highlights
- Analyzed **2.3M Sales**, **286.4K Profit**, and **37.9K Quantity Sold**
- Visualized profit distribution across **49 states**
- Compared profitability by:
  - Region
  - Category
  - Ship Mode
  - Quarter & Year
- Identified top-performing cities and states based on profit contribution
- Built interactive filters for dynamic business analysis

## 📈 Key Insights
- **Technology** category generated the highest profit contribution
- **West Region** showed the strongest overall profitability
- **Standard Class Shipping** contributed the highest share of profit
- Profit growth showed a steady upward trend from **2014 to 2017**

This dashboard helps stakeholders monitor business performance, track profitability trends, optimize shipping strategies, and support data-driven decision-making through interactive visual analytics.
